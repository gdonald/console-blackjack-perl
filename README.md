## Install from CPAN

    cpanm Console::Blackjack

### Run

    console-blackjack.pl

## Install from Github

    git clone https://github.com/gdonald/console-blackjack-perl.git
   
### Run

    cd console-blackjack-perl/Console-Blackjack

    perl -Ilib script/console-blackjack.pl
